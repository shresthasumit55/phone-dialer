About This Project

Testing of the magnifier

I used the magnifier provided by the Windows operating system. It allows magnifying the screen for different levels. I used the magnifier to create a repository in github and adding collaborators to the project. It took a bit of time getting around the screen with the magnifier switched on. It was difficult to scroll through the pages at first. From my experience I think the magnifier is not a tool you would use if you are comfortable with the normal size of the contents on screen. It is not comfortable to use the computer in a magnified mode if you are okay with the normal mode (100 % zoom). This means the zoom level should not greater or less than the level you are comfortable with. But for people with low vision, the magnifying feature is very helpful since it magnifies the content to the size they can easily see. Some amount of time is required to get used to scrolling, navigating with the magnifier switched on. The screenshots of my experience with magnifier can be found in link below:
https://drive.google.com/open?id=1lwkUmYIVyb7Hncb77PdBW7YC9xJ7MiGN

Features added to the site for low vision users

To assist the users with low vision, zooming feature was added to the application. The users who are unable to see the contents of the page in its normal size can use this feature to view the contents in a more zoomed in format. Two buttons are added which increases and decreases the size of the contents and different users can set the size to their own preference. For now an upper limit has been set for the zoom level beyond which the contents cannot be zoomed.

Along with the customizable content size, a high contrast mode was also added to the site. Some users find it difficult to read the text in the screen because they cannot easily differentiate the background color with the color of the contents in the front. So a button is provided to switch the high contrast mode on and off. The high contrast mode sets the background color to black and uses other colors to the content to create a high contrast with the background. This will be helpful to users having color blindness as well.
